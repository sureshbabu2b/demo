This is my Git learning file

Sequencial Commands

1. Git clone - to copy the repository to local desktop code example: git clone https://github.com/sureshbabu2b/demo.git

2. Git add - to add the changes before commit

3. Git commit - to commit the changes in local desktop to server code example: git commit -m "changed the code"

4. Git push - to push the changes to the server

5. Git pull - to pull the latest modified to local desktop

6. Git checkout - to create the new branch code example: git checkout -b "new_branch" //new_branch is the name of the new branch to be created

6. Git merge - to merge the branch to main or vice versa code example: in the main, git merge new_branch

Non Sequencial Commands

1. Git status - to view the status of the file

2. Git branch - to view the list of branches and the branch you are in
